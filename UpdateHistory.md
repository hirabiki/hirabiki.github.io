## ExposurePlus Version History

### 1.1.1 - 2026/03/25

* Improved auto exposure handling in brightly-lit worlds
* Fixed the issue of Local Contrast being too weak in dark areas
* Increased Local Contrast strength and improved the rendering of highlights

### 1.1.0 - 2026/03/24

ExposurePlus is now less of a hassle to use as the one major usability hurdle has been mitigated. It can be used in mirrors without having to manually re-enable auto exposure functionality.

* Fixed the exposure flickering issue when the workaround is performed
* Implemented an automatic workaround as a result
* Fixed the issue when avatar scale is small or large
* Reorganized and simplified the main menu of ExposurePlus
* Added a button to reset all adjustments to default values
* Changed the instructional messages to reflect changes to the workaround
* Improved image quality of Local Contrast
* Tweaked and fine-tuned the the dynamic tone mapping algorithm to improve out-of-the-box lighting with default settings

### 1.0.2 - 2026/03/21

* Made it so that the bounding box reported becomes minimal
* Reduced minimum double-press speed from 0.5s to 1.0s
* Improved version update checking timing

### 1.0.1 - 2026/03/21

* Removed extraneous `using` that could cause script error for some users

### 1.0.0 - 2026/03/20

* Initial release
